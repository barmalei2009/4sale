RECOMMENDED IMAGE SPECS
------------------------
  Format  : JPEG (.jpg) or WebP (.webp) for best compression
  Size    : 800 × 600 px minimum (landscape or square works best)
  Quality : 80–85 JPEG quality is sufficient for web
  Ratio   : 4:3 or 1:1 recommended (the shop crops to fill the card)

WHAT HAPPENS WITHOUT IMAGES
-----------------------------
If an image file is missing or fails to load, the shop automatically
shows a beautiful SVG iris placeholder silhouette. The shop works
fully without any images — just add them when you have photos.

TIPS
----
- Use free iris photos from the American Iris Society (irises.org)
  or your own garden photography.
- Run images through https://squoosh.app to compress before uploading.
- WebP is supported by all modern browsers and is ~30% smaller than JPEG.
  To use WebP, update the "image" field in products.json accordingly
  (e.g. "images/immortality.webp").
